# The Schikko Rules Ledger

> An ancient tome for a modern age. A real-time, collaborative web application for tracking rules, decrees, and, of course, punishments.

## Features

- Real-time collaborative ledger powered by Firebase Firestore.
- Secure Schikko role for administrative tasks with an annual reset.
- The Scribe's Logbook with search and filtering for the last 30 days.
- Google Calendar integration to display upcoming events.
- Dynamic decrees with full editing and reordering.
- Enhanced ledger functionality with statistics, searching, and sorting.
- Oracle's Judgement (Gemini AI) for thematic consequence suggestions.

---

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript (ES Modules)
- Styling: Tailwind CSS (Play CDN for development)
- Backend & Database: Google Firebase (Firestore, Cloud Functions, Anonymous Auth, Hosting)
- AI: Google Gemini API
- Deployment: GitHub Actions for CI/CD
